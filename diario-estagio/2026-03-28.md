### ⚠️ Dificuldades encontradas

- Diferença entre:
  - dados por substância (Secção 3)
  - dados do produto como um todo (Secção 9, 14, 15)

- Outputs inconsistentes do LLM:
  - às vezes lista
  - às vezes objeto
  - necessidade de normalizar parsing

- Problemas de integração:
  - mismatch entre schema e pipeline
  - erros como:
    - `'list' object has no attribute state`
    - `unexpected keyword argument`

- Problemas de ambiente:
  - instalação de dependências (fitz / PyMuPDF)
  - necessidade de usar virtual environment
